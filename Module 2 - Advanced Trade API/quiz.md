# Module 2: Knowledge Check

### Q1: Which protocol should be used for receiving real-time price updates with the lowest latency?
A) REST API (Polling)
B) WebSocket API
C) Email Alerts
D) GraphQL

### Q2: You receive a `403 Forbidden` error when trying to place a buy order. What is the most likely cause?
A) The system is down.
B) You are out of ETH.
C) Your API Key permissions do not include "Trade" scope.
D) You are using a public Wi-Fi.

### Q3: Advanced Trade API Keys (CDP) use which authentication technology?
A) Basic Auth (Username/Password)
B) OAuth 1.0
C) JWT (JSON Web Tokens) signed with a Private Key
D) API Token in the URL query string

### Q4: If you encounter a `429 Too Many Requests` error, what should your script do?
A) Retry immediately and faster.
B) Wait a fixed amount of time (e.g., 1 second) and retry.
C) Implement "Exponential Backoff" (wait 1s, then 2s, then 4s...).
D) Create a new API key to bypass the limit.

---
**Answers:**
1: B
2: C
3: C
4: C
