## Getting Started

### 1) Create and activate a Python virtual environment (macOS/Linux)

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 2) Install dependencies

```bash
pip install -r requirements.txt
```

### 3) Explore the course

- Start with `course outline.md` for a structured view of the curriculum
- Open a module directory to follow its README and complete the exercises

## Notes

- Do not commit secrets. Keep credentials in a secure store.
- Module-specific environment variables, setup steps, and scripts are documented within each module's README.
